hexo操作

hexo操作

  1. hexo help # 查看帮助
  2. hexo version #查看Hexo的版本
  3. hexo algolia # 更新search庫
  4. hexo new “postName” #新建文章
  5. hexo new post “title” # 生成新文章：\source_posts\title.md，可省略post
  6. hexo new page “pageName” #新建页面
  7. hexo clean #清除部署緩存
  8. hexo n == hexo new #新建文章
  9. hexo g == hexo generate #生成静态页面至public目录
  10. hexo s == hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
  11. hexo d == hexo deploy #将.deploy目录部署到GitHub
  12. hexo d -g #生成加部署
  13. hexo s -g #生成加预览

部署

每次部署的步骤，可按以下两部来进行

  1. hexo cl
  2. hexo d -g
